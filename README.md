🧩 Aventura no Labirinto

Projeto em Python onde o jogador precisa encontrar a saída de um labirinto evitando paredes.

🎮 Funcionalidades :

Movimento com teclado (W, A, S, D)

Sistema de pontuação

Geração aleatória de labirinto

Resolução automática (recursividade)

▶️ Como executar : python main.py --name Henrique --use-rich

🕹️ Controles
W → Cima
S → Baixo
A → Esquerda
D → Direita

🏆 Objetivo : Chegar até o ponto final F sem bater nas paredes.

📌 Tecnologias:
Python
Biblioteca Rich (visual)

📚 Observação : Projeto desenvolvido para atividade acadêmica.
